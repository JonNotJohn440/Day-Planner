# Day-Planner

## User Story

```md
AS AN employee with a busy schedule
I WANT to add important events to a daily planner
SO THAT I can manage my time effectively
```

## Usage

```md
In order to use the Day-Planner, simply click anywhere in the planner and begin typing you note.
Once you are done, click the save icon and the note will be saved to your localStorage 
```
## Build
```md
This app runs in the browser and features dynamically updated HTML and CSS powered by jQuery. 
The [Moment.js](https://momentjs.com/) library was also used to work with date and time.
